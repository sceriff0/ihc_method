Purpose

Cross-modality validation of the IHC readout against bulk-RNA marker genes: if IHC protein positivity is real, the fraction of cells positive for a marker should track the bulk-RNA expression of that marker’s gene across patients. This is the most direct check — one protein against its own transcript — and needs no deconvolution model.

For each IHC marker we take its canonical gene(s) (marker_gene_map), average multi-gene markers, and correlate bulk expression vs the per-patient fraction of <marker>_sign == "+" cells.

Per-patient bulk expression of each marker’s gene(s)

expr_long <- counts_data |>
  mutate(patient_id = norm_id(Sample)) |>
  pivot_longer(-c(Sample, patient_id), names_to = "gene", values_to = "expr")

# marker -> its genes, one row per (marker, gene). Explicit $-access + strsplit
# (no NSE) so it is robust to the base pipe / tidyr separate_rows() quirks.
gene_rows <- do.call(rbind, Map(function(m, g)
  data.frame(marker = m, gene = trimws(strsplit(g, ",")[[1]]), stringsAsFactors = FALSE),
  marker_gene_map$marker, marker_gene_map$genes))

missing <- setdiff(gene_rows$gene, unique(expr_long$gene))
if (length(missing)) message("genes absent from RNA matrix: ", paste(missing, collapse = ", "))

# keep only genes present in the RNA matrix
marker_genes <- gene_rows |> filter(gene %in% unique(expr_long$gene))

# Per patient, average expression across a marker's available genes.
marker_expr <- expr_long |>
  inner_join(marker_genes, by = "gene", relationship = "many-to-many") |>
  group_by(marker, patient_id) |>
  summarise(rna_expr = mean(expr), .groups = "drop")

Sensitivity: the mean marker z-score (continuous staining intensity) instead of the positive-cell fraction, in case a hard _sign threshold loses signal.

ihc_z <- ihc_marker_fraction(ihc_data) |>

  pivot_longer(-patient_id, names_to = "marker", values_to = "ihc_z") |>

paired_z <- marker_expr |>

Notes

  • Expect strong positive correlation for lineage-specific markers (PTPRC/CD45, CD3, CD8A, CD163, NCAM1/CD56) where protein and transcript co-vary; weaker or absent for markers regulated post-transcriptionally or with poor antibody specificity — a low rho flags an IHC channel worth auditing.
  • PANCK maps to EPCAM + keratin genes (averaged); SMA→ACTA2, CD56→NCAM1, PDL1→CD274, PD1→PDCD1, FSP1→S100A4. wrongL1CAM is excluded by construction.
  • Correlations are cross-patient and small-n; every row carries its n.
